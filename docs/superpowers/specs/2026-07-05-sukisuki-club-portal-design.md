ステータス: 承認済み(ユーザー指示によりデプロイまで自律実行)

## 目的

sukisuki.club のルートドメインに、「蓮ノ空のこと好き好きクラブのみなさん」(蓮ノ空女学院スクールアイドルクラブのファンネーム)のためのファンサイトポータルを公開する。

- 主目的: サブドメイン等で運用中のファンサイトへの導線
- 副目的: 蓮ノ空を知らない訪問者への簡単な紹介
- 重点: **これまでの103期〜105期(第1章)を大切にする**。106期(第2章)は「これから」として控えめに触れるのみ(ユーザー指示)

## デザイン

コンセプト: **「第1章の一日」**。ページ全体をひとつの空に見立て、スクロールで朝(始まり)→夕暮れ(卒業)→夜(第2章を待つ)へ降りていく。年表が夜に沈み、最後に星(106期)が瞬く。

- 配色: 朝空 #d8edf9 → 昼 #a9b4e0 → 薄暮 #6f5b9e → 夜 #100c24。セクションごとにグラデーションを連結し継ぎ目なく遷移
- メンバーカラー11色はチップ・ユニット帯として使用
- 書体: Shippori Mincho B1(見出し・明朝) × Zen Kaku Gothic New(本文) — Google Fonts
- 和の要素: 縦書きセクションラベル(writing-mode: vertical-rl)、半透明の washi 風カード
- 装飾は自作SVG(蓮の花・雲・星)のみ。公式画像・ロゴは一切使用しない
- アクセシビリティ: focus-visible、prefers-reduced-motion 対応、モバイル対応

## 技術構成

- 純静的サイト(ビルドなし): `public/index.html` + `public/styles.css` + `public/favicon.svg`
- Cloudflare Workers 静的アセット配信(mypick と同方式)
- `wrangler.jsonc`: `routes: [{ pattern: "sukisuki.club", custom_domain: true }]`
- GitHub Actions: main への push で `wrangler deploy`(要 `CLOUDFLARE_API_TOKEN` シークレット)
- リポジトリ: `Suki-Suki-Club/sukisuki-club-portal`(public)

## 実装手順

1. 静的ファイル実装(index.html / styles.css / favicon.svg)
2. wrangler.jsonc / package.json / README / deploy workflow
3. ローカルで `wrangler dev` 起動し表示確認
4. GitHub org へリポジトリ作成・push
5. `wrangler deploy` で本番公開、https://sukisuki.club 応答確認
